# samuelpatrone.com

Personal website for Samuel Patrone — phenomenological physicist, PhD candidate at Caltech, and incoming postdoctoral researcher at UC San Diego.

## Live Site

[samuelpatrone.com](https://samuelpatrone.com)

## Structure

```
index.html    — Single-page site (About, Research, Publications, Teaching, Beyond Physics, Contact)
style.css     — Dark cosmos theme with glassmorphism and scroll animations
script.js     — Particles.js config, scroll reveals, nav interactions, counter animations
assets/       — Profile photo
CNAME         — Custom domain config
```

## Tech

- Plain HTML / CSS / JS — no build step
- [particles.js](https://github.com/VincentGarreau/particles.js/) for interactive background
- [Font Awesome](https://fontawesome.com/) + [Academicons](https://jpswalsh.github.io/academicons/) for icons
- [Inter](https://rsms.me/inter/) via Google Fonts
- Hosted on GitHub Pages
